TipFC

Description


TipFC is a hockey analytics visualizer. By retrieving data from the NHL API, a group member and I are able to display select stats for any NHL player or team in the 2019/20 season.

The project is developed with Javascript, HTML/CSS in the front end, and PHP, SQL in the backend.
